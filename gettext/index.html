Library gettext

This library exposes the following toplevel modules:

    Library gettext.base

    This library exposes the following toplevel modules:

    • Gettext Modules to use in libraries and programs.
    • GettextCategory Functions to manipulate category.
    • GettextCompat Gettext compatibility with the native gettext API
    • GettextConfig
    • GettextConfigDune
    • GettextDummy Concrete implementation based on nothing.
    • GettextFormat Check string equivalence regarding printf use.
    • GettextMo
    • GettextModules
    • GettextTypes Types and exception of ocaml-gettext.
    • GettextUtils Misc. utilities.

    Library gettext.extension

    This library exposes the following toplevel modules:

    • GettextCharset Signature of module for charset conversion
    • GettextCompile Functions for extraction/compilation/installation of PO/MO file.
    • GettextLocale Implements different operation over locale/category.
    • GettextPo
    • GettextRealize Module type for the function realize.
    • GettextTranslate Signature of module for translation storage / access.
